## 自己PR
Webサイトやアプリケーションのデザイン/UI設計から最終的な実装まで、ほぼすべての工程に携わることが可能です。デザインからフロントエンド実装までを得意領域としています。
HTMLマークアップ仕様書や[WordPressの構築設計ガイドライン](https://qiita.com/kgsi/items/65ee2d7814e5cfa199d2)などのアウトプット多数あります。

## 興味
プロダクト開発全般、その中でも特にUI設計とそれに付随するフロントエンド開発に興味があります。
プログラムはJavaScriptをメインとしていますが、PHPやPythonも対応可能です。学習が必要な案件も柔軟に対応します。

## 得意なこと
- グラフィックデザイン
- UIデザイン
- Webフロントエンド実装
- ハイブリッドアプリケーション実装

## 得意でないこと・できないこと
- 本格的なサーバサイド・インフラ開発（BaaSに丸投げが多いです）
- 本格的なゲーム開発
- 本格的なネイティブアプリケーション開発
- ルーチン作業
