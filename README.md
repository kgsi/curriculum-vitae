## 自己PR
所属している会社の特徴で代理店を通さず、直で受注する案件が多くを占めるため、必然的にWebサイトやアプリケーションのデザイン/UI設計から、最終的な実装まで全ての工程で携わり、ほぼ一通りの制作経験を積むことができました。今現在は顧客対応まで対応範囲が広がり、ワンストップに業務を担当しています。

また開発部に所属してからは、実装ルールがバラバラだったシステム構築のルールを統一する目的で、HTMLマークアップ仕様書や[WordPressの構築設計ガイドライン](https://qiita.com/kgsi/items/65ee2d7814e5cfa199d2)などをまとめ、社内レベル、及び社外への発注レベルを一定値以上に底上げする、などの啓蒙活動も行っています。
